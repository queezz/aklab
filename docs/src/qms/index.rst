QMS
===============
QMS - Quadrupole Mass Analyzer |QMS wiki|.

We use Ulvac's |Qulee BGM| spectrometer. 

Software for QMS control and data convertion |Qulee QSC| is only avaliable for Windows.
It is not freely distributed, but we have images of several older versions.
If a new version is required, we need to request Ulvac directly.
